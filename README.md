# Institute of Digital Risk (IDR)

The **Institute of Digital Risk (IDR)** is an industry-led training and deployment institute focused on digital, cyber, and technology risk roles in regulated, high-consequence environments.

## Project Overview

This project is a professional, responsive single-page website designed to establish a strong brand identity for IDR. It bridges the gap between academic insight and real-world industry practice.

### Key Features

- **Single-File Architecture**: The entire website (HTML, CSS, and JavaScript) is contained within a single `index.html` file for maximum portability and performance.
- **Modern Branding**: Features a custom-designed geometric logo symbolizing structure, risk, and resilience.
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.
- **Dark Mode Support**: Includes a toggle for light and dark themes with persistent user preferences.
- **Interactive Elements**:
    - Scroll progress indicator.
    - Smooth-scrolling navigation.
    - Interactive FAQ accordion.
    - Functional contact form simulation.
- **Tech Stack**: Vanilla HTML5, CSS3 (using Flexbox and Grid), and modern JavaScript (ES6+). Icons provided by Lucide.

## Design Choices

- **Typography**: Uses *Space Grotesk* for technical, bold headings and *Inter* for clean, professional body text.
- **Color Palette**: A high-contrast palette of Vibrant Orange (#F27D26), Deep Black, and Soft White to convey a minimalist tech/education feel.
- **Symbolism**: The cube-inspired logo represents the structured approach to managing complex digital risks.

## Mandatory Sections Included

1. **Hero**: Clear mission statement and primary call-to-actions.
2. **About IDR**: Explanation of the industry-led model and UK University partnerships.
3. **Service Pillars**: Academy, Innovation & Incubation, and Advisory Services.
4. **Pipeline Model**: Visual representation of the Train – Hire – Innovate – Deploy workflow.
5. **Community**: Overview of the ecosystem (Financial Services, Critical Infrastructure, etc.).
6. **FAQ**: Answers to common inquiries.
7. **Contact**: Registration of interest form.
